## 關鍵架構決策（勿忘）

1. **防迴圈**：`if message.author.bot: return` ❌ **絕對不能用這個**
   正確寫法：`if message.author.id == self.user.id: return`，其他 bot 用 `[hop:N]` 計數
2. **Discord 而非 Telegram**：Telegram bot 無法收到其他 bot 訊息（平台硬限制）
3. **SQLite WAL**：三個 bot 行程 + dashboard 共用 `/opt/addwii-pm/data/addwii.db`
4. **claude -p**：Max 訂閱，零 API 費用；**不設 ANTHROPIC_API_KEY 全域環境變數**
5. **所有 .py 檔**：code-writer → code-qa（sequential），QA PASS 才 commit
